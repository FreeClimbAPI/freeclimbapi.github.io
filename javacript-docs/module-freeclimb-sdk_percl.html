# inner outDial(to, from, actionUrl, callConnectUrl, optionsopt) → {Object}

Build an OutDial PerCL command.
Parameters:
Name Type Attributes Description
to string The E.164 representation of the target phone number.

callConnectUrl string The URL which will be requested informing the result of the OutDial. PerCL is expected if the call status is {@code module:freeclimb-sdk/enums/callStatus.IN_PROGRESS}.
